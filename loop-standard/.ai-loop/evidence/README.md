# Evidence Directory

Each phase gets one subdirectory:

```text
evidence/<phase-id>/
  prompt.md
  report.md
  diff.patch
  verify.log
  status.txt
```

Codex must treat missing files or `MISSING:` placeholders as blocking evidence
defects.

Use `.ai-loop/templates/prompt.md` and `.ai-loop/templates/report.md` as the
human-readable templates for the first two files. The diff, verification log,
and status files are generated by `Collect-LoopEvidence.ps1`.
